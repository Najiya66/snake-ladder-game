# 🐍 Snake & Ladder Game

An interactive Snake & Ladder game built using **HTML, CSS, and JavaScript** with smooth animations and turn-based gameplay.

---

## 🎮 Features

- 🎲 Dice rolling system
- 🔴🔵 Two-player mode
- 🐍 Snake (slide down)
- 🪜 Ladder (climb up)
- 🚶 Smooth step-by-step movement animation
- 🔄 Turn switching system
- 🎯 Win detection
